# Nutlet.Utility
一个不需要第三方依赖的工具库

## Install
1. 打开`Window/Package Manager`
2. 点击左上角加号，选择`Add package of git URL`
3. 输入`https://github.com/ResidualSoils/nutlet.utility.git?path=Assets/Nutlet/Utility`
